# Support & help

## Getting help

- 💬 **[Discord](https://discord.gg/WWHDU62WW)** — setup help, usage questions, and hanging out with
  other Commanders. The fastest place to get unstuck.
- 🐛 **[GitHub Issues](https://github.com/dseelinger/CovasPlusPlus/issues)** — report a bug or
  request a feature. The bug form asks for your `check_setup.py` output — run
  `.venv\Scripts\python.exe check_setup.py` (it's change- and cost-free) and paste the result; it
  usually pinpoints the problem.
- 📖 **These docs** — install, setup, and a page per feature. Use the search box up top.
- 📓 **[Changelog](https://github.com/dseelinger/CovasPlusPlus/blob/main/CHANGELOG.md)** — what
  changed between versions, so you can decide whether to update.

!!! note "One-person project"
    COVAS++ is solo-maintained, so replies are best-effort and asynchronous — issues and messages
    are read, but a response can take a few days. A clear report (with `check_setup.py` output) is
    the fastest path to a fix. See the maintainer-status note in the
    [README](https://github.com/dseelinger/CovasPlusPlus#maintainer-status--response-expectations).
